== Description ==

Woo Products Tree is a simple plugin which adds a widget displaying navigation tree of products and categories on your WooCommerce store.

Widget opens current product category and highlights viewing product there. Allows customer open and close other categories in tabs and directly go to the item of interest.

Widget can shows or hide price, product image, put images to the right or to the left side from the products list. For best site color combination widget provides adjustment its main color.

Only two levels of categories inserts available in current free version.
If you like and need in this product, please support us how it possible, then we will continue creation. Thank You!

== Installation ==

1. Upload `woo-products-tree` to the `/wp-content/plugins/` directory. Normally it is possible to do in WordPress 'Plugins' menu by pressing 'Install Now' button.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the widget tab and add widget 'METEO-IS. Woo Products Tree' to the bar panel.
4. Make a necessary adjustments or leave as default and press save button.
5. Check your site.

== Frequently Asked Questions ==

= How to return default widget options? =
Just delete widget from sidebar, then add it again.

= How to disable background color on widget tabs? =
Set "Tab color transparency" to 0.
